01 Headless Browser · Scraping
h4ckf0r0day / obscura
Navegador headless open source escrito en Rust, pensado para scraping a escala y para agentes IA.
18.9k ★ · 1.3k forks Apache 2.0
En una frase
Un reemplazo drop-in para headless Chrome que arranca al instante, consume ~30 MB y trae anti-detección y servidor MCP incluidos.
Funcionalidades
  • Servidor Chrome DevTools Protocol compatible con Puppeteer y Playwright.
  • Comandos fetch, serve y scrape en paralelo con workers.
  • Modo stealth: fingerprint por sesión y bloqueo de 3.520 trackers.
  • Servidor MCP integrado con 12 tools para agentes.
Características
  • V8 embebido directamente en Rust: JavaScript real.
  • Consumo 30 MB vs 200+ MB de Chrome, arranque instantáneo.
  • Binario 70 MB y Docker distroless de ~57 MB.
  • Renderiza páginas en 85 ms frente a ~500 ms.
Stack
Rust 99.8% V8 embebido CDP · Puppeteer · Playwright Docker distroless
github.com/h4ckf0r0day/obscura

02 AI Coding Usage · macOS Menu Bar
steipete / CodexBar
App diminuta para la barra de menú de macOS que muestra el uso y los límites de tus proveedores de coding con IA.
14.2k ★ · 1.1k forks MIT License
En una frase
Deja de adivinar cuánto te queda: cada proveedor de IA que uses aparece en la barra de menú con su cuota, coste y cuenta atrás al reset.
Funcionalidades
  • Soporte para 40+ proveedores: Codex, Claude, Cursor, Gemini, Copilot, Grok...
  • Ventanas de sesión, semanal y mensual con countdown al reset.
  • Balances de créditos, gasto de Admin API y escaneo local de coste.
  • Merge Icons mode: agrupa proveedores en un solo icono.
Características
  • Reutiliza sesiones ya existentes: OAuth, device flow, API keys, cookies.
  • Sin passwords almacenados en ningún lado.
  • CLI incluida para scripts y CI en macOS y Linux.
  • Widgets de WidgetKit y polling de incidencias del proveedor.
Stack
Swift 6.2+ macOS 14+ Sonoma CLI · WidgetKit Homebrew · AUR
github.com/steipete/CodexBar

03 General AI Agent Framework
FoundationAgents / OpenManus
Framework Python para construir agentes IA generales, alternativa abierta al Manus original sin invite code.
56.4k ★ · 9.8k forks MIT License
En una frase
Escribes tu idea en la terminal, el agente descompone la tarea y usa navegador, editor y APIs para ejecutarla de principio a fin.
Funcionalidades
  • Un solo comando python main.py para arrancar.
  • Modo run_flow con multi-agente y DataAnalysis Agent.
  • Modo run_mcp para exponer el agente vía MCP.
  • Automatización de navegador con Playwright.
Características
  • LLM configurable vía config.toml: cualquier endpoint compatible con OpenAI.
  • Instalación con uv o conda, entorno Python 3.12.
  • Construido sobre anthropic-computer-use, browser-use y crawl4ai.
  • Publicado con DOI en Zenodo por el equipo de MetaGPT.
Stack
Python 97.8% Playwright · MCP uv · Docker config.toml BYO LLM
github.com/FoundationAgents/OpenManus

04 Omnichannel Chatbot Platform
ChatbotXIO / ChatbotX
Chatbot omnicanal open source para flujos agénticos, alternativa self-hosted a Wati, ManyChat, Chatfuel y Respond.io.
17 ★ · v0.2.2 AGPL-3.0 + Commercial
En una frase
Un docker-compose y tienes tu propio ManyChat con inbox unificado, flow builder, agentes IA, broadcasts y APIs/CLI/MCP para automatizar.
Funcionalidades

05 Disposable Email · Self-Hosted
kasteckis / TempFastMail
Bandeja de entrada de correo temporal, self-hosted, para recibir emails desechables sobre tu propio dominio.
328 ★ · 41 forks GPL-3.0
En una frase
Un docker-compose sobre tu dominio y un Email Worker de Cloudflare gratis: tienes tu propio 10minutemail privado.
Funcionalidades
  • Recepción de emails con vista integrada (HTML y texto).
  • Retención automática de 48 horas por email.
  • Auto-eliminación al vencer el TTL y borrado manual desde admin.

Características

  • Runtime moderno con FrankenPHP y Symfony.
  • Frontend TypeScript sobre plantillas Twig.

07 Cloud Proxy Provisioning
claffin / cloudproxy
Provisiona proxies HTTP en varios cloud providers para esconder la IP de tus scrapers, controlado por API.
1.7k ★ · 107 forks MIT License
En una frase
Le pasas tokens de tu cloud y arranca proxies bajo demanda con IPs distintas, los mantiene sanos y los recicla sin downtime.
Funcionalidades
  • 5 cloud providers: DigitalOcean, AWS, GCP, Hetzner, Vultr.
  • REST API en localhost:8000 con Swagger docs.
  • UI Vue.js para escalar arriba/abajo y ver salud de proxies.
  • Multi-cuenta por provider con regiones y sizes distintos.
Características
  • Rolling deployments: reciclado con cero downtime.
  • Pool fijo con auto-scaling entre MIN y MAX.
  • Endpoint /random para obtener un proxy al azar.
  • Aviso del autor: datacenter IPs son bloqueadas por muchos sitios.
Stack
Python 80% · FastAPI Vue.js UI Docker · PyPI DO · AWS · GCP · Hetzner · Vultr
github.com/claffin/cloudproxy
