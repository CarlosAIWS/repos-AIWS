01 Email Agent · Edge Runtime
cloudflare / agentic-inbox
Un cliente de email autoalojado con agente IA, corriendo entero sobre Cloudflare Workers.
Cloudflare Oficial Apache 2.0
Qué es
  • Un cliente de email completo con agente IA integrado.
  • Corre 100% en el edge de Cloudflare.
  • Un Durable Object por buzón con su propia base SQLite.
Para qué sirve
  • Recibir, leer y responder emails con un agente que borrador siempre, envío con tu OK.
  • Buscar conversaciones por contenido o contexto.
  • Adjuntos en R2, sin servidor que mantener.
Características
  • 9 herramientas de email para el agente.
  • Servidor MCP incluido para conectar Claude Code o Cursor.
  • Deploy con un solo click en tu cuenta Cloudflare.
Stack
Cloudflare Workers Durable Objects · R2 Email Routing Workers AI · Kimi K2.5 React 19 · Hono Agents SDK
github.com/cloudflare/agentic-inbox

02 Integration Platform
NangoHQ / nango
Construye integraciones de producto con IA. 700+ APIs unificadas, runtime de producción.
7.4k ★ · Replit, Ramp, Mercor Elastic License
Qué es
  • Una plataforma open source para integraciones de producto.
  • Tres primitivas: Auth · Proxy · Functions.
  • El AI builder genera funciones TypeScript desde lenguaje natural.
Para qué sirve
  • OAuth gestionado para 700+ APIs sin programarlo.
  • Sincronización de datos para RAG e indexación.
  • Tool calling y MCP listos para tu agente.
Características
  • Self-hostable o cloud, SOC 2 · HIPAA · GDPR.
  • Compatible con Cursor, Codex, Claude Code, LangChain.
  • Código legible y editable, no caja negra.
Stack
TypeScript Docker CLI + API MCP · LangChain · Pydantic AI 700+ APIs

03 Social Media Platform
brightbeanxyz / brightbean-studio
Gestión de redes sociales open source. La alternativa libre a Buffer, Sendible y SocialPilot.
10+ plataformas sociales AGPL-3.0
Qué es
  • Una plataforma completa de gestión de redes sociales.
  • Publica en Facebook, Instagram, LinkedIn, TikTok, YouTube y 5 más.
  • APIs first-party directas, sin agregador en medio.
Para qué sirve
  • Programar, publicar y aprobar contenido en varias cuentas.
  • Bandeja unificada con análisis de sentimiento.
  • Portal de cliente con magic links de 30 días.
Características
  • Multi-workspace, RBAC, kanban de ideas, audit log.
  • Repo companion con agente IA que lo controla por REST y MCP.
  • Ojo licencia AGPL-3.0 si lo metes en SaaS.
