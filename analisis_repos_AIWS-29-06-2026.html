01 Launch Video Skill
latent-spaces / brag
Una skill de Claude Code que convierte el proyecto que acabas de crear en un vídeo de lanzamiento corto y compartible.
Claude Code skill MIT-ish (Hyperframes)
En una frase
Tu proyecto entra como código, sale como vídeo con música, movimiento y copy listo para postear.
Funcionalidades
  • Comando /brag dentro de cualquier proyecto.
  • Genera plan, brief, copy para compartir y brag.mp4.
  • Permite ajustar el tono: --tone "fake Serie A de 2016".
  • Música y SFX bundle incluido.
Características
  • La skill decide la narrativa: producto, tono y momentos.
  • Renderizado delegado a Hyperframes (HeyGen).
  • Marketplace de plugins: /plugin install.
  • Output en carpeta local brag-output/.
Stack
Claude Code · Plugin Node.js 22+ FFmpeg Hyperframes CLI
github.com/latent-spaces/brag

02 Local Project Memory
raiyanyahya / recall
Plugin de Claude Code que mantiene memoria persistente del proyecto entre sesiones, totalmente en local y sin llamadas a APIs.
Claude Code plugin MIT License
En una frase
Claude empieza cada sesión en frío: este plugin guarda el contexto en local para que retome donde lo dejaste.
Funcionalidades
  • history.md: log append-only de cada sesión.
  • context.md: resumen reciente al iniciar.
  • Comandos: /recall:save, /recall:show, /recall:log.
  • Hooks automáticos SessionStart, Stop, SessionEnd.
Características
  • Sin API key, sin LLM, sin nube: nada sale del equipo.
  • Resumen con TF-IDF + TextRank vendorizado.
  • Detección y redacción de secretos antes de escribir.
  • Confina escrituras al proyecto: cero rutas absolutas.
Stack
Python 3.9+ (stdlib) NumPy opcional Git diff/log integrado CodeQL · Bandit · Ruff
github.com/raiyanyahya/recall

03 AI Writing Cleanup
blader / humanizer
Skill que elimina las huellas típicas del texto generado por IA y lo deja sonar a una persona real.
25.2k ★ · 2.4k forks MIT License
En una frase
Pegas un texto con olor a IA y te devuelve una versión más natural, con tu estilo si le das ejemplos.
Funcionalidades
  • Detecta 33 patrones de escritura IA.
  • Inflación de relevancia, vocabulario IA, em-dashes, listas vacías…
  • Voice calibration: replica tu estilo con muestras.
  • Doble pasada: borrador + auditoría "obviously AI".
Características
  • Compatible con Claude Code y OpenCode.
  • Basado en la guía de Wikipedia "Signs of AI writing".
  • Sin instalación: solo un SKILL.md.
  • Ejemplos before/after para cada uno de los 33 patrones.
Stack
Claude Code Skill OpenCode Skill Markdown puro Sin dependencias
github.com/blader/humanizer

04 Browser Side Panel
abundantbeing / hermes-browser-extension
Extensión de Chrome/Edge que abre un side panel conectado a un runtime local de Hermes Agent (Nous Research).
12 ★ · Alpha pública MIT License
En una frase
No es otro chatbot del navegador: usa los modelos, skills y MCPs ya configurados en tu Hermes Agent local.
Funcionalidades
  • Side panel Chrome MV3 con la Side Panel API.
  • Captura título, URL, pestañas, texto seleccionado.
  • Mete el HTML como contexto no fiable antes de mandarlo.
  • Sesión persistente local con streaming de respuestas.
Características
  • Conexión a 127.0.0.1:8642, todo local.
  • Solo lectura: no clica, no rellena, no envía.
  • Sin permisos de cookies, history, downloads.
  • Páginas restringidas: banca, pagos, gobierno.
Stack
Chrome MV3 · Edge JavaScript Hermes Agent (Nous) Side Panel API

05 Agent Control Plane
duncatzat / vigils
Capa local de control entre tus agentes IA y las herramientas que tocan: ver, aprobar y proteger.
129 ★ · v0.1.7 Apache 2.0
En una frase
Un control plane que se mete entre tu agente y el resto: ves qué hace, apruebas lo importante y mantienes los secretos a salvo.
Funcionalidades
  • Audit ledger SHA-256 hash-chained con búsqueda full-text.
  • Cola de aprobación para acciones destructivas o sensibles.
  • Redacta 13+ tipos de credenciales y PII antes de mandar.
  • Sandbox fail-closed: Wasm + Linux Landlock.
Características
  • Default-deny firewall con DSL de políticas en Rust.
  • Gateway MCP en stdio y HTTP con pinning de tools.
  • App desktop Tauri 2 + Vue 3 con replay de sesiones.
  • Extensión Chrome que redacta antes de pegar en ChatGPT, Claude, Gemini.
Stack
Rust · 15 crates Tauri 2 · Vue 3 Chrome MV3 Wasm · Landlock LSM
